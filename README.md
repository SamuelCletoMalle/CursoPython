# Python – Curso completo desde cero (Ejercicios y proyectos)

Este repositorio contiene ejercicios, ejemplos y proyectos desarrollados durante la realización del curso **“Curso completo de Python: Programación en Python desde cero”**.

El objetivo de este repositorio es documentar mi aprendizaje de Python mediante práctica progresiva, aplicando los conceptos del lenguaje a través de ejercicios y pequeños proyectos. Este repositorio forma parte de mi formación en programación y también sirve como portfolio técnico.

---

## Temas del curso

**Tema 0 – Configuración del entorno**
Instalación de Python, configuración del entorno de desarrollo (IDE) y primeros pasos ejecutando programas.

**Tema 1 – Introducción a Python**
Conceptos básicos del lenguaje, sintaxis inicial, variables y primeros programas.

**Tema 2 – Operadores en Python**
Uso de operadores aritméticos, lógicos, de comparación y de asignación.

**Tema 3 – Tipos de datos complejos**
Uso de estructuras de datos como listas, tuplas, diccionarios y conjuntos.

**Tema 4 – Estructuras de control**
Uso de condicionales (`if`, `elif`, `else`) y bucles (`for`, `while`) para controlar el flujo del programa.

**Tema 5 – Programación orientada a objetos**
Creación de clases y objetos, atributos, métodos y conceptos básicos de programación orientada a objetos.

**Tema 6 – Trabajando con objetos**
Uso avanzado de clases y objetos, incluyendo organización del código y métodos especiales.

**Tema 7 – Programación modular**
Organización del código en módulos para mejorar la estructura y reutilización del programa.

**Tema 8 – Manejo de excepciones**
Gestión de errores en Python mediante estructuras como `try` y `except`.

**Tema 9 – Proyecto: Videojuego Snake**
Desarrollo del clásico videojuego Snake aplicando los conocimientos adquiridos durante el curso.

**Tema 10 – Proyecto: Filtro de detección de SPAM**
Proyecto de clasificación de mensajes utilizando técnicas básicas de Machine Learning.
